# 내 취향대로 가입하는 모임 어플 "Wetting"   
회원가입 할 때 입력한 취향 기반으로 모임을 추천해주고 가입할 수 있는 어플
### 기능
+ 메인 - 추천 모임, 내 모임
+ 회원관리 - 회원가입, 로그인, 마이페이지
+ 모임 관리 - 모임 생성, 가입, 탈퇴
+ 검색 - 일반 검색, 지역 검색
### 개발 환경
+ node.js
+ aws
+ mariaDB
